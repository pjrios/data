Pre-activities · Part 2 · About 10 minutes

Is this the correct use?

Use the examples above. Decide whether each display is a good choice for the question.

Bar chart We want to compare the number of students in four clubs. Is a bar chart a good choice?

Line chart We want to compare favorite colors, which have no time order. Is a line chart the best choice?

Pie chart We want to show how temperature changes each day for two weeks. Is a pie chart a good choice?

Scatter plot We want to see whether study hours and quiz scores are related. Is a scatter plot a good choice?

Table We need to find each student’s exact score. Is a table a good choice?
