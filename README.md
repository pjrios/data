# Data Detectives

A static 15–20 minute data-cleaning practice activity for students. It includes 12 dataset rows, eight intentional problems, a change log, progress tracking, data-care questions, and a teamwork reflection.

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js` to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.

No backend or build process is required.
