Part 1 · Learn · About 4 minutes

Learn before you clean

Read these ideas and examples. They will help you make safe corrections during the investigation.

About 4 min
1

Why clean data?

Incorrect or inconsistent data can produce misleading results and unfair decisions.

2

What should you notice?

Look for blanks, repeated rows, impossible values, spelling differences, and inconsistent units or formats.

3

How should you correct it?

Use evidence, make values consistent, and record every change. Never invent missing information.

Common errors and safe corrections

Blank cell

Example: an empty study time

Safe action: write Not provided when the answer is unknown.

Repeated row

Example: the same student, time, and subject appear twice

Safe action: mark the repeated row so it is not counted twice.

Impossible number

Example: -20 min or 900 hours

Safe action: use Not provided unless a source confirms the correct value.

Spelling or case

Example: Scince or math

Safe action: standardize them to Science and Math.

Missing unit

Example: 45 in a study-time column

Safe action: add the expected unit when the table makes it clear, such as 45 min.

Format difference

Example: 1 hour among values written in minutes

Safe action: use one consistent format, such as 60 min.

Remember: inspect the whole row, correct only what the evidence supports, and document what changed and why.

Part 2 · Practice · About 13–16 minutes

Your mission

Work through all five stages. Aim for careful decisions, not speed.

  1. Inspect: Read the entire dataset before changing anything.
  2. Clean: Fix blank cells, repeated rows, impossible numbers, spelling differences, and missing units.
  3. Document: Select the issue and save each corrected row.
  4. Review: Check the cleaned dataset and complete the reflection.
  5. Submit: Download your evidence report and upload it to Google Classroom.
Data-care rule: Never invent a missing answer. Use Not provided when the correct value is unknown.
