15–20 Minute Data Cleaning Practice

Data Detectives

Find data problems, make careful corrections, and explain your decisions.

Your mission

Work through all four stages. Aim for careful decisions, not speed.

Suggested time: 15–20 min
  1. Inspect: Read the entire dataset before changing anything.
  2. Clean: Fix blank cells, repeated rows, impossible numbers, spelling differences, and missing units.
  3. Document: Select the issue and save each corrected row.
  4. Review: Check the cleaned dataset and complete the reflection.
Data-care rule: Never invent a missing answer. Use Not provided when the correct value is unknown.
